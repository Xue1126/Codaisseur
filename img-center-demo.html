There are several ways to fit an image in a div:

  1. You can set the width and height, but it'll be strethed.
  2. You can set the width OR the height, but it'll have either space left or be too big.
  3. You can do the above, but with overflow: hidden, but it only works if you know whether the image aspect ratio is bigger or smaller than the container one.
  4. You can just make it very big and use overflow: hidden - you'll lose a lot of image and it'll be inflexible.
  5. You can make it a background image, and set background-size: cover; - works but it'll be a background image
  6. You can use object-fit: cover; (info) - the perfect way! but only in modern browser versions
